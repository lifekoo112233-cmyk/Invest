# Grid Trader 📊

**그리드 트레이딩 백테스트 & 계산기 PWA**

## 기능
- 📂 **CSV 업로드** - 어떤 자산이든 (나스닥, 골드, 외환, 원유...)
- 📈 **백테스트** - 월별 히트맵, 자산차트, 출금내역
- 🧮 **계산기** - 청산시뮬, 역산계산, 평균단가, 반등수익
- 📱 **PWA** - 폰 홈화면에 앱처럼 설치 가능

## 배포 방법

### Vercel (권장)
1. GitHub에 push
2. vercel.com → New Project → GitHub 연결
3. 자동 배포 완료!

### 로컬 실행
```bash
npm install
npm start
```

## CSV 형식
Investing.com 역사적 데이터 CSV 그대로 사용 가능
- 나스닥 100, 골드, EUR/USD, 원유 등 모든 자산 지원

## 폰에 설치
1. 배포된 링크를 폰 브라우저에서 접속
2. Android: 메뉴 → 홈화면에 추가
3. iPhone: 공유 → 홈 화면에 추가
